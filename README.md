# AI Todo Chatbot Phase III

Your AI-powered todo chatbot is ready to use! 

## Quick Start

Run these two commands in separate terminals to start both servers:

### Backend Server (port 8000)
```bash
cd backend
uvicorn main:app --reload
```

### Frontend Server (port 3000)
```bash
cd frontend
npm run dev
```

Once both servers are running, open your browser to http://localhost:3000 to start chatting with your Todo Bot!
