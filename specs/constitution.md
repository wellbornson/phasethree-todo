# Project Constitution

## 1. Mission
Build a clean, sophisticated, and fast AI Todo Bot using the Agentic Dev Stack.

## 2. Architecture Principles
- **Backend**: Stateless FastAPI.
- **Database**: SQLModel with Neon PostgreSQL.
- **Integration**: MCP Server (Official SDK) + OpenAI Agents SDK.
- **Frontend**: OpenAI ChatKit (React/Vite).

## 3. Code Standards
- **Python**: PEP 8, Type Hints (Pydantic/SQLModel).
- **Frontend**: Functional React, Hooks, TypeScript.
- **Testing**: Pytest for backend, Vitest for frontend.
