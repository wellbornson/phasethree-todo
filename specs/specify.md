# Specification: AI Todo Bot (Phase III)

## 1. Overview
An intelligent task management system where users interact via a chat interface to manage their todos.

## 2. Features
- **Natural Language Task Management**: Add, list, update, complete, and delete tasks via chat.
- **Persistent Storage**: Tasks stored in Neon PostgreSQL.
- **Agentic Logic**: Powered by OpenAI Agents SDK and MCP.

## 3. User Stories
- As a user, I want to say "Add buy milk" and have it saved.
- As a user, I want to ask "What do I need to do?" and see my list.
- As a user, I want to mark tasks as done via conversation.
